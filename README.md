# PowerShell Scripts

Collection of PowerShell scripts focused on:

- IT Support
- Windows Administration
- Automation
- Infrastructure Studies

## Scripts

### info-sistema.ps1
Displays detailed Windows system information.

### network-info.ps1
Displays network and IP configuration information.

### disk-usage.ps1
Displays disk and storage usage information.

## Technologies
- PowerShell
- Windows 10/11
- Networking
- IT Support
